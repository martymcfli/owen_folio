Key Features

  • Interactive Heat Map

    Visual culture score representation across companies with hover tooltips and filtering by industry, location, and size.

  • Salary Benchmarking

    Compare salaries across 1,600+ data points with filters for role, level, and company. Includes percentile breakdowns.

  • Anonymous Reviews

    Submit and read anonymous company reviews with sentiment analysis and helpfulness voting.

  • AI-Powered News

    OpenAI integration fetches and summarizes recent company news, providing context for culture changes.

  • Job Openings

    Integrated job listings for each company so users can act on their research immediately.

Tech Stack

Built with modern, scalable technologies for performance and maintainability.

React TypeScript tRPC Drizzle ORM PostgreSQL OpenAI API Tailwind CSS Recharts Framer Motion Supabase

Architecture

Frontend: React with TypeScript, styled with Tailwind CSS and Framer Motion animations.

Backend: tRPC for type-safe API calls, Drizzle ORM for database operations.

Database: PostgreSQL via Supabase with full relational schema for companies, reviews, salaries, and user data.

AI Integration: OpenAI API for news summarization and sentiment analysis.

Business Impact

Culture Heat Map addresses a critical gap in the job search process. Job seekers often accept offers without understanding workplace culture, leading to poor fit and high turnover. By aggregating culture scores, salary data, reviews, and news into one platform, candidates can make data-driven decisions about their careers. For companies, it provides transparency that attracts culture-fit candidates and reduces hiring friction.
